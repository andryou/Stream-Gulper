# Stream Gulper
Download streaming video easily, and optionally grab subtitles (if available) as well.

Coded in NSIS/Batch and uses the following projects: youtube-dl, Livestreamer, Subtitles Edit, ffmpeg

## To Use
1. Run StreamGulper.exe
2. Paste in a URL to a video you want to download
3. Choose where you want to save the video (and subtitles, if option enabled)
4. Click on Download
